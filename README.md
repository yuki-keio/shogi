# 将棋Web

GPLv3ライセンスのWeb将棋ゲームです。

🔗URL： **[https://shogi.yuki-lab.com/](https://shogi.yuki-lab.com/)**

## 機能

- AI対戦：初級〜伝説級（有段者レベル）
- 対人戦モード
- 駒の動き方ガイド
- PWA対応（オフラインプレイ可能）

## 技術スタック

- JavaScript
- YaneuraOu (WASM) - 高難易度AI用
