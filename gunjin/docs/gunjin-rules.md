# 軍人将棋 1人用Webゲーム 採用ルール

## 盤面

- 見た目は 6 列 x 9 行。
- `C1-D1` は敵司令部、`C9-D9` は自司令部だが、ゲーム内部ではそれぞれ 1 ノードとして扱う。
- 可動駒は司令部への出入りができ、前面 2 マスに加えて左右隣のマスからも出入りできる。
- 中央帯の `A5/C5/D5/F5`（川） は進入不可。
- `B5/E5` は橋の通路で、駒は止まれない。地上駒はこの橋からのみ渡河できる。

## 駒構成

- 1 枚: 大将 / 中将 / 少将 / 大佐 / 中佐 / 少佐 / 騎兵 / スパイ / 軍旗
- 2 枚: 大尉 / 中尉 / 少尉 / 地雷 / 工兵 / タンク / 飛行機

## 移動

- 大将 / 中将 / 少将 / 大佐 / 中佐 / 少佐 / 大尉 / 中尉 / 少尉 / スパイ: 前後左右 1
- タンク / 騎兵: 前 1-2、後 1、横 1
- 工兵: 縦横に任意距離、飛び越し不可
- 飛行機: 縦に任意距離、横 1、川や駒の飛び越し可
- 地雷 / 軍旗: 不動

## 戦闘

- 階級は `大将 > 中将 > 少将 > 大佐 > 中佐 > 少佐 > 大尉 > 中尉 > 少尉`
- 同種は相打ち
- 飛行機は将官にのみ負ける
- タンクは将官 / 飛行機 に負ける
- 騎兵は工兵 / スパイにのみ勝つ
- 工兵は地雷 / スパイにのみ勝つ
- スパイは大将にのみ勝つ
- 地雷は工兵 / 飛行機に負け、それ以外とは相手もろとも除去される
- 軍旗は直後方 1 ノードの味方駒の強さを借りる。背後が地雷なら軍旗と攻撃駒のみ除去
- AI側もプレイヤー側も、相手の駒の種類は勝敗がつくまで見えない。自分の駒しかわからず、相手の駒は推論するのみ

## 勝敗

- 全ての可動駒が敵司令部に入れるが、`大将 / 中将 / 少将 / 大佐 / 中佐 / 少佐` のいずれかで占領した時だけ勝ち
- 相手の可動駒を全滅させても勝ち
- 同一戦闘で双方の可動駒が尽きた場合は引き分け
